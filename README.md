# Macroeconomía II - Notas de Clase

Notas de clase del curso de Macroeconomía II, Universidad Católica del Norte.

## Estructura

```
clases/
  clase02_aspa_keynesiana.tex          # Clase 2: Aspa Keynesiana
  clase03_multiplicador_IS.tex         # Clase 3: Multiplicador y Curva IS
  clase05_LM_equilibrio_ISLM.tex       # Clase 5: Preferencia por la Liquidez y Equilibrio IS-LM
  clase10_regla_de_oro.tex             # Clase 10: La Regla de Oro en el Modelo de Solow
guias/
  guia02_politica_fiscal.tex           # Guía 2: Política Fiscal
```

## Compilación

Cada archivo `.tex` se puede compilar de forma independiente con `pdflatex`:

```bash
cd clases
pdflatex clase02_aspa_keynesiana.tex
```

## Contenido

### Clase 2 - Aspa Keynesiana
1. Notación Matemática
2. Aspa Keynesiana (supuestos, demanda agregada, componentes)
3. Funciones del Modelo (consumo, ahorro, inversión, gasto gubernamental)
4. Función de Demanda Agregada
5. Determinación de la Producción de Equilibrio
6. Estática Comparativa

### Clase 10 - La Regla de Oro en el Modelo de Solow
1. Función de producción Cobb-Douglas
2. (a) Comprobación de los supuestos: rendimientos marginales decrecientes, rendimientos constantes a escala, condiciones de Inada, esencialidad
3. (b) Función de producción per cápita
4. (c) Capital de Estado Estacionario y estática comparativa
5. (d) Capital de la Regla de Oro (problema del planificador, condición PMK = δ)
